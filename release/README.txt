FLIR Batch Parameter Editor

This package contains the standalone Windows executable for batch-editing
FLIR radiometric JPEG camera parameters.

Files:
- FLIR Batch Parameter Editor.exe

Quick use:
1. Open the executable.
2. Choose a folder or select files.
3. Tick the files you want to update.
4. Leave any field blank to keep that parameter unchanged.
5. Click Execute Batch Changes.

Notes:
- Optional .bak backups can be enabled inside the app.
- The tool edits FLIR CameraInfo metadata inside supported radiometric JPEGs.
